# Football AI:
This project is used to track players and the football with other features implemented as well.
## Files:
- The file ObjectDetection.ipynb is used for training a YOLO Object Detection Model.
- The file PitchDetection.ipynb is used for training a YOLO Key-Point Detection Model.
- The file FootballAI.ipynb is used to run these models and add other features.
## Roboflow API Key:
To run theses files on Kaggle, you will need to add your private Roboflow API Key in the Secrets under the label 'Roboflow'
Add-onns -> Secrets

Note: Also go on Settings -> Accelorator -> GPU P100.
